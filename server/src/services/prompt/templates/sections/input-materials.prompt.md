# Input Materials

The input of this task may contain:

- chart image — the procedure chart page(s)
- tabular image — the procedure tabular description page(s)
- coordinate image — the waypoint coordinate page(s)
- minima image — the minima table page(s)
- support summary — structured summaries of airport metadata, runway data, navaids,
  communications, chart index and other supporting sections

How to use each material:

- Chart pages: recognize drawn geometry, tracks, labels, arrows, DME arcs, radials,
  lead radials, MSA, holding patterns and every text bound to the drawing.
- Tabular pages: recognize leg sequence, path terminator, course, distance, altitude,
  speed and turn direction as procedure legs.
- Coordinate pages: recognize waypoint identifiers and their coordinates.
- Supporting summaries: context and cross-checking ONLY — they are NOT procedure content
  by default and must pass the support-filtering rules before entering the output.
